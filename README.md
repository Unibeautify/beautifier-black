> Black beautifier for Unibeautify

## Installation

```bash
npm install --global @unibeautify/beautifier-black
```

Then change `.unibeautifyrc.yml`:

```yaml
Python:
    beautifiers: ["Black"]
```

## Usage

See [`unibeautify-cli`](https://github.com/Unibeautify/unibeautify-cli) for details.
